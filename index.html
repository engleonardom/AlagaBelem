Política de Privacidade do AlagaBelém

1. Informações Coletadas

O aplicativo AlagaBelém não coleta quaisquer dados pessoais, sensíveis ou identificáveis. Todas as funcionalidades operam localmente no dispositivo.

2. Finalidade do Uso das Informações

As informações processadas pelo aplicativo são utilizadas exclusivamente para exibir dados relacionados a alagamentos em tempo real, sem envio para servidores externos.

3. Compartilhamento de Dados

O AlagaBelém não compartilha dados com terceiros e não utiliza serviços externos de análise de comportamento ou publicidade.

4. Segurança dos Dados

Embora o app não armazene informações, implementamos medidas de segurança para proteger quaisquer dados processados localmente. Dados, como localização, são descartados imediatamente após o encerramento do uso.

5. Retenção e Exclusão de Dados

Como o aplicativo não armazena dados, não há necessidade de retenção ou exclusão. Caso futuras funcionalidades exijam coleta de dados, garantiremos opções claras para exclusão.

6. Direito dos Usuários

Você tem o direito de saber como suas informações são tratadas. Entre em contato conosco para esclarecer qualquer dúvida sobre privacidade.

7. Uso de Cookies

O aplicativo não utiliza cookies ou tecnologias semelhantes. Se no futuro forem adicionadas tecnologias que envolvam cookies, você será informado e terá a opção de recusá-los.

8. Alterações na Política de Privacidade

Esta política poderá ser atualizada para atender a novas exigências legais ou técnicas. Notificaremos os usuários sobre alterações significativas.
